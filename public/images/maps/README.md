# Mapa de España

Coloca aquí un archivo PNG transparente de España con el nombre `spain-outline.png`.

El archivo debe ser:
- Formato: PNG con transparencia
- Dimensiones recomendadas: 1000x700px o proporción similar
- Contorno de España sin fondo (transparente)

Puedes descargar un mapa de España desde:
- https://www.flaticon.com/free-icon/spain_206671
- https://simplemaps.com/resources/svg-maps
- O cualquier otra fuente de mapas SVG/PNG de España
